# Chat-Box
簡易聊天機器人，檔案分有 Server後端 和 Web前端 兩部分 <br>
Server部分內嵌簡單的回覆程式 <br>
Web部分是前端三兄弟 <br>
Buliding a WebSocket Chat Room through Node.JS <br> 
with a little chat bot server which is able to produce some simple reply
